\doxysubsection{Detailed Description}
A class that manages shaders and shader programs. 

The \doxylink{classivf_1_1_shader_manager}{Shader\+Manager} class provides functionality to load, compile, and link shaders, as well as manage shader programs. It allows loading shaders from files or strings, and provides methods to retrieve the current shader program and apply it. The class also provides error checking for shader compilation and linking.
